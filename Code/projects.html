Digital Airline Web-Service

This application is the backend code for a hypothetical airline. The functions of the program for a simple user are as follows:

  • Registration
  • Login/Logout
  • Search for flights with filters
  • Search for a specific flight
  • Ticket reservation
  • Find all the reservations a user has
  • Find details for a specific reservation
  • Cancel a reservation
  • Delete accout

In addition, if the user is an admin they can do the following additional operations:

  • Create a flight
  • Renewal of flight ticket prices
  • Delete a flight
  • Find more details about a specific flight

More information can be found in the Github repository here.

Data Base with Java

This project is an application that makes use of a database. More specifically, it is about a company and the functions it can do are as follows:

  • Exit program.
  • Insert and update queries.
  • Retrieve the list of all the company's warehouses.
  • Retrieve the list of all parts by receiving lot and by storage bin in each warehouse.
  • Retrieve the available capacity and the remaining capacity of each bin in each warehouse of the company.
  • Update the available capacity and the remaining capacity of each bin in each warehouse of the company
  • Retrieve the list of all employees of the company by manager
  • Retrieve the list of names of all the company's managers
  • Retrieve a list of components that are "potential" parts of assembled individual parts/components.
  • Retrieve the list of all "re-orders" of parts stored in the system, sorted by the surname and name of the manager.
  • Delete for a company employee a specific contact phone number assigned in the database

More information can be found in the Github repository here.
